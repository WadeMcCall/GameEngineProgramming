**************************
Dependencies:: edit the build_game.bat file to set the paths of your dependencies.
you will need: Crazy Eddie GUI, Ogre 1.8.1, boost, freeimage, zlib, TDM-GCC mingw compiler, tinyXML, OIS, BASS,lua, and bullet physics
**************************

Left analog stick moves your character around
Right analog stick rotates him.
esc on keyboard exits game.
jump button makes your character jump
background music is from Super Smash Brothers Melee - Final Destination theme
